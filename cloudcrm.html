Objective

Design the Clean Professional CRM dashboard to manage the workflow, employees, and organization.

Project Goals

We conducted the stakeholder interview along with project manager to understand what are the stakeholder goals.

They need a dashboard to manage their projects and clients.

They want to maintain the projects and clients easily, which should be easily usable.

This dashboard has two authenticated access one is user and the other will be Admin.

Industry / Competitive Analysis

I have collected the competitors tools and analysed the features and brainstormed with our team to improve usablity and user engagement and to improve the better management.

We searched for the existing project management tools and came up with some inspirations.

While all the pages are more complicated with their tools. I stumbled on what feature should i copy from it.

I asked my leads for help and found out that there are websites that provide templates for free and also paid. One of them is ThemeForest

So I searched for the existing project management dashboard analyzed the user flow that is commonly practiced and came up with some inspirations and ideas.

Key insights from the analysis

Users will be required to provide login details, necessitating a login form.

Projects will be listed along with their associated tasks, and updates on project progress will be provided.

As multiple people will handle projects simultaneously, a chat box for communication and a file upload area are necessary.

A dashboard page will display overall analytics for all ongoing progress.

The employees and clients pages will feature listings with detailed information, similar to the project page.

I've also proposed implementing a Kanban board for task management.

A pricing page will be accessible after login.

User Flow

I took a small note on what would be the features/functions that will be made by the user. Categories the features with the name and planned it into creation of each page

Project Goals

Problems

  • The Kanban implementation on the existing pages.
